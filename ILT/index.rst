---------------
Getting Started
---------------

This workshop is for teaching containers based on a Linux operating system. The O/S that is being used is based on the RancherOS (https://rancher.com/rancher-os/). This Linux version is very small and has the docker-engine and docker-composer default installed.

Environment support
+++++++++++++++++++

- Cluster:
	
	- HyperVisor: AHV
	- AOS: version 5.8 or higher
	- Calm: version 5.9 or higher

- PRIM Central: version 5.9 or higher

Instructions to create the workshop
+++++++++++++++++++++++++++++++++++

To have the attendees get access to their testing environment the easy way, use the below blueprint and import it into Calm.

:download:`Calm Blueprint <blueprint/containers101.json>`

After import, change the blueprint to your environment and launch it.
